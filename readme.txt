== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Go to the wp-admin/plugin-install.php
1. Add New Plugin by searching Softinn Check Availability Widget and install
1. Activate the plugin through the 'Plugins' menu in WordPress
1. After activate go to wp-admin/widgets.php drag Softinn Check Availability Widget to the sidebar

== Frequently Asked Questions ==

= How this widget work? =

Hotel owner need to sign up Softinn Booking in http://mysoftinn.com/ 
After that just need to key in your ID then the widget will let user to check hotel owner room availability
